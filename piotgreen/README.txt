디렉터리: piotgreen
담당자: 선지훈

이 디렉터리는 PiotGreen 프로젝트의 웹 애플리케이션을 포함하고 있으며, Spring Boot를 활용하여 개발되었습니다. 각 시스템(침입 감지, 관수, 광량 제어)의 데이터를 통합 관리하고 사용자 인터페이스를 제공하는 역할을 합니다. 또한, 사용자가 모든 기능을 웹 페이지를 통해 실시간으로 모니터링하고 제어할 수 있도록 구성되었습니다.

주요 역할:
- 각 기능별 데이터를 통합하여 LCD 화면 및 웹 인터페이스에 실시간 표시
- 사용자에게 직관적인 제어 화면을 제공하여 시스템 상태 확인 및 설정 조정 가능
- Spring Boot를 활용한 데이터 관리 및 통신 기능 구현
