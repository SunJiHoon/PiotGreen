디렉터리: irrigation_system
담당자: 정용희

이 디렉터리는 토양 수분 모니터링 및 자동 관수 시스템 코드를 포함하고 있으며, 농작물 재배에 필요한 적정 수분을 유지하는 역할을 합니다. 토양 수분 센서를 통해 실시간으로 데이터를 수집하고, 설정된 임계값 이하로 수분이 감소하면 자동으로 관수 시스템이 작동하도록 구현되었습니다.

주요 역할:
- 토양 수분 상태를 실시간 모니터링하여 농작물의 생장 환경을 유지
- 자동 관수 기능을 통해 사용자의 부담 감소
- 수분 상태와 관수 기록을 데이터베이스에 저장하여 일정 관리에 활용 가능
